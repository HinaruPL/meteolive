# MeteoLive

Modern Polish weather radar website with live maps for storms, rainfall, wind, temperature and weather alerts.

## Project goal

MetoLive is planned as a fast, mobile-friendly and SEO-focused weather website for Poland.  
The site will include live weather maps, radar sections, city pages and educational articles explaining how to read weather data.

## Planned sections

- Storm radar
- Rain radar
- Wind map
- Temperature map
- Weather alerts
- Weather for Polish cities
- Guides about weather maps and radar data

## Tech stack

- HTML
- CSS
- JavaScript
- Static hosting
- Cloudflare Pages
